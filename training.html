Training

There are a number of ways you can train your employees in Game Dev Tycoon. On this page we will list all of them.

Book Studies

These training methods are available from when you first start in your new office.

Training Cost Benefit
Game Dev Gems 5, 15K ++ Design, ++ Technology, + Research, + Speed
Make Me Think! 5, 15K +++ Research,+ Technology,  + Design, + Speed
Game Design for Pirates 5, 15K +++ Design, + Technology, + Research
Don't Repeat Yourself 5, 15K +++ Speed,+ Research, + Technology,  + Design
Code Incomplete 5, 15K +++ Technology, + Design, + Research

G3 courses

Training Cost Benefit
G3 Pixel Cup 10, 70K + Design
G3 Code Jam 10, 70K + Technology
G3 Game Jam 10, 70K + Design, + Technology
G3 Time Trials 10, 70K + Speed (Large Amount), + Research (Small Amount)
G3 Innovation Challenge 10, 70K + Research
